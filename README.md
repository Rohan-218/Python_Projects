# Python Projects

This repository contains a collection of beginner-friendly Python scripts, examples, and small projects. It is intended for learners who want to practice Python fundamentals such as data types, control flow, file handling, functions, classes, and basic automation.

## Contents

- `E-mail_verification.py` / `Email_verification_Regex.py`
  - Email validation scripts using direct string checks and regular expressions.
- `Voice_Assistant.py` / `Voice_Assistant_.py`
  - Simple voice assistant projects using speech recognition and text-to-speech.
- `Excel_project/tranasctions.py`
  - Example project for working with Excel data and charts.
- `Sqlite/roster.py`, `Sqlite/tracks.py`
  - SQLite-based data processing examples.
- Utility and learning scripts:
  - `list.py`, `Tuple.py`, `Dictionary.py`, `kwargs.py`, `args.py`, `exception.py`, `file_handling.py`, `inheritance.py`, `comprehensions.py`, `class.py`, and more.
- `bs4/` and `soupsieve/`
  - Included packages for HTML parsing and CSS selectors.

## How to Use

1. Open the repository in your Python editor or IDE.
2. Activate your Python environment.
3. Run individual scripts with a command like:
   ```powershell
   python .\script_name.py
   ```
4. Install any required libraries if a script depends on external packages.

## Notes

- This repository is a learning workspace and includes both standalone scripts and package folders.
- Some files may reference external libraries such as `speech_recognition`, `pyttsx3`, `pyjokes`, `openpyxl`, or `sqlite3`.
- Review each script before running it to ensure any required data files or dependencies are available.
